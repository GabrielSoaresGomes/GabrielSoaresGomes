## 👨‍💻 Sobre mim

Sou Backend Engineer com experiência no desenvolvimento de APIs, sistemas distribuídos, plataformas SaaS, mensageria e aplicações em tempo real.

Atuo principalmente com Node.js/TypeScript, construindo soluções backend com foco em performance, resiliência, escalabilidade e integração entre serviços.

Também tenho experiência com Java e Python, e atualmente estou direcionando minha evolução técnica para Go, com foco em concorrência, microsserviços, sistemas performáticos e aplicações cloud-native.

Tenho vivência com tecnologias como RabbitMQ, Socket.IO, PostgreSQL, Redis, Docker, AWS, observabilidade e arquitetura orientada a eventos.
